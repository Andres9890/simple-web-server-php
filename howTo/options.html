Basic options

Option Name Value Default Description
Enabled 
enabled
 true/false Whether the server is turned on or not
Directory 
path
 file path Directory to serve files from
Port 
port
 number 1 - 65535 8080 Port that server is accessible on
Accessible on local network 
localnetwork
 true/false false Makes the web server accessible over LAN to other computers on the network. Access using this computer's local IP address.

Basic rules

Option Name Value Default Description
Automatically show index.html 
showIndex
 true/false true Automatically serve index.html file when no file path is specified
Single page rewrite (for SPA) 
spa
 true/false false Automatically rewrite all paths that don't match files to index.html in root directory. For Single Page Applications.
Show directory listing 
directoryListing
 true/false true Show a list of files in the specified directory instead of a 404 page
